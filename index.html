Configure o Firebase para jogar em multiplayer

⚙️ Configuração Rápida (gratuito)

  1. 1Acesse console.firebase.google.com → clique em "Criar Projeto"
  2. 2No projeto criado: Build → Realtime Database → Criar banco de dados. Escolha "iniciar no modo de teste"
  3. 3Configurações do Projeto → Seus apps → Web. Registre o app e copie o firebaseConfig
  4. 4Cole os valores nos campos abaixo e clique em Salvar & Jogar

As configurações são salvas no seu navegador (localStorage)
